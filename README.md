# 本地歌词提供器 (LocalLyricProvider)

为词幕 (Lyricon) 提供本地歌词支持。自动读取音频内嵌歌词（LYRICS 标签）或扫描指定目录下的同名 .lrc 文件，并智能解析路径，适配多种播放器。

## 功能特性

- **内嵌歌词优先**：自动读取音频文件中的 LYRICS/USLT 标签，支持增强型 LRC（逐字时间轴、多角色、翻译）。
- **本地 .lrc 文件扫描**：默认扫描 `/storage/emulated/0/Lyrics`、`/Music`、`/Download` 及用户自定义目录，自动匹配同名歌词。
- **智能路径解析**：通过 MediaStore 多级匹配（精确 → 模糊 → 仅标题 → 文件名），解决部分播放器无法直接提供文件路径的问题。
- **避免歌词闪退**：内嵌歌词读取成功后，不会被后续的空搜索结果覆盖。
- **支持增强型 LRC**：包括逐字时间轴、多角色、翻译、元数据等。

## 支持的播放器（已经测试过）

- PowerAmp（广播 + SAF 路径）
- 椒盐音乐 (Salt Music)
- 海贝音乐 (HiBy Music)
- UAPP (USB Audio Player Pro)
- QQ音乐（QQ Music）
- 网易云音乐（仅限本地音乐）
- 酷狗音乐 HD（仅限音乐）
- 酷我音乐（仅限本地）
- 波点音乐（仅限本地）
- QQ音乐 HD（仅限本地）
- ColorOS 自带播放器（仅限本地）

## 安装与使用

1. 安装 Xposed 框架（如 LSPosed）并激活。
2. 安装 **词幕 (Lyricon)** 应用。
3. 安装本模块，并在 LSPosed 中启用模块，在**作用域**中勾选需要显示歌词的播放器（如 `com.salt.music`）。
4. 授予播放器**存储权限**（如果需要扫描外部 .lrc 文件）。
5. 重启系统界面（或重启手机），打开播放器播放音乐即可看到状态栏歌词。

## 注意事项

- 如果播放器本身自带歌词功能（如光锥音乐），建议关闭，避免冲突。
- 内嵌歌词读取依赖 **TagLib** 库，支持 MP3、FLAC、M4A 等常见音频格式。
